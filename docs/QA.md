# Memory Pager — QA 테스트 명세서

> 커플(2인) 전용 폐쇄형 소셜 앱. 사진·손낙서를 삐삐 감성으로 주고받고, 함께 키우는 펫이 낙서를 보고 그림 일기를 그려준다.
> Flutter(Android) + FastAPI + Socket.IO + MySQL + GPU(EXAONE LLM / SD 1.5 + LoRA).
>
> **이 문서는 실제 코드(front-remake)를 화면별로 정독해 작성한 QA 테스트 스펙이다.** 각 요소가 **실서버에 연동되는지 / 로컬 세션 한정인지 / 정적 목업(死버튼·하드코딩)인지**를 정직하게 구분한다. FEATURES.md 의 화면별 표는 일부 낡았으므로, 화면 동작의 최종 근거는 이 문서다.
>
> 작성 기준: `front-remake` @ FCM/LoRA 반영 이후. 대상 앱 패키지 `org.madcamp.memory_pager`.

---

## 0. 범례 · 판정 기준

**연동 구분**
- 🟢 **실서버 연동** — 실제 백엔드 API/소켓을 호출한다(엔드포인트 명시). 실서버 모드(`API_BASE` 지정)에서만.
- 🔵 **로컬·세션 한정** — 서버 API가 (아직) 없어 세션 로컬 상태로만 동작한다. **설계상 의도**된 것(앱 재시작 시 리셋). 데모로는 동작하나 영속·양방향 동기화는 안 됨.
- ⚪ **정적·미구현** — 死버튼(핸들러 없음)·하드코딩 목업·no-op 스낵바. QA 시 "동작 안 하는 게 정상"으로 취급.

**우선순위**
- P0 핵심 플로우(등록·낙서·일기·실시간) / P1 보조 기능 / P2 연출·장식

**실행 환경 표기**
- `[웹]` 웹 데모(mock)로 검증 가능 · `[실서버]` 실서버 부팅 필요 · `[2인]` 커플 2계정 필요 · `[실기기]` 안드로이드 실기기 필요(FCM·홈위젯)

## 1. 테스트 환경 준비

| 항목 | 값 |
|---|---|
| 실서버 API | `https://anjonghwa.madcamp-kaist.org` (터널, VPN 불필요) |
| 헬스체크 | `GET /v1/health` → `{"status":"ok","db":"ok","gpu":"ok"}` |
| 완성 APK | GitHub Actions `build-apk` 아티팩트 `memory-pager-apk` (실서버 연결 빌드) |
| 웹 데모(mock) | `flutter run -d chrome` (API_BASE 없음 → 온보딩 완료 상태로 홈 시작) |
| 웹(실서버) | `flutter run -d chrome --dart-define=API_BASE=https://anjonghwa.madcamp-kaist.org` |
| 2인 세팅 | 기기 2대, 또는 브라우저 2개에 `--dart-define=DEVICE_UID=couple-a` / `couple-b` |

**듀얼모드 원리:** 앱은 `mock.real`(= `api != null`) 하나로 갈린다. `API_BASE`가 있으면 부팅 때 `register→/me→그룹 로드`로 실서버에 붙고, 없으면 목 데이터 데모다. 아래 🟢 항목은 **실서버 모드에서만** 서버에 반영되고, 데모에선 같은 UI가 로컬로만 동작한다.

**신규 그룹의 빈 상태가 정상:** 갓 만든 그룹은 낙서 0·일기 0·레포트 0이다. 일기는 펫 활동이 쌓이고 자정 배치가 돌아야 생긴다.

## 6. 권장 스모크 순서 (데모 시연용, 실서버 2인)

1. A: 등록 → 그룹 생성 → 초대코드(설정에서 복사) 공유
2. B: 등록 → 코드로 참여 → 별명 설정
3. A→B 낙서 전송 → B 홈/앨범에 실시간 등장(소켓) + (앱 내리면) 푸시
4. B 뷰어에서 답장 낙서 → A에 스레드로 도착
5. 사라지기 낙서 → 확인 5초 후 양쪽 소멸
6. 오늘의 질문 양쪽 답변 → 상태 동기화
7. 펫 쓰다듬기 5회 → 서프라이즈, 찌르기 왕복
8. (일기 축적 후) 그림일기·월간 레포트 확인
9. 손그림 20장 축적 또는 `POST /groups/{id}/style/train` → learned 화풍 일기

> P0(1~6·낙서/실시간/일기)이 데모의 핵심. P2(달력·이웃집·서프라이즈 연출)는 정적 목업임을 감안해 시연.
