## 두드림 — 커플의 낙서로 키우는 폐쇄형 공간

안녕하세요, 커플 앱 **‘두드림’**으로 돌아온 **안종화, 이종혁**입니다.

혹시 커플 앱을 왜 쓰는지 고민해보신 적 있나요? 저희가 물어보니 답이 대체로 두 가지로 좁혀졌습니다.

1. **둘만의 공간** — 일정·답변·사진을 기록해두고 종종 들여다보며 만족감을 느낀다.
2. **지속적인 소통** — 매일 질문을 주고받으니 자꾸 대화하게 되고, 굳이 궁금하지 않았던 것도 알게 된다.

그래서 저희는 **이 두 가지를 정확히 짚는 앱**을 만들었습니다. 화면을 보며 설명드릴게요.

---

## 앱 둘러보기

### 1. 시작 — 그룹 만들기 / 입장

앱을 처음 실행하면 **그룹을 생성하거나 초대 코드로 가입**할 수 있습니다. 상대방이 그룹에 들어오면 그때 앱에 진입합니다. 그룹 **정원은 2명**, 커플만의 닫힌 공간입니다.

```
📷 docs/screens/01_onboarding.png   (준비 중 — 라이브 셋업 때 캡처 예정)
```

### 2. 홈 화면

앱에 들어오면 처음 보이는 화면입니다. 가장 먼저 눈에 들어오는 건 마스코트 **모리**, 그리고 핵심 기능인 **낙서 보내기** 버튼입니다. 함께한 날짜(D-day), 콕 찌르기, 오늘의 질문, 방금 도착한 낙서 알림까지 한 화면에 모여 있습니다.

```
📷 docs/screens/02_home.png
```

### 3. 월간 레포트

서로 주고받은 낙서와 활동을 모아 **매달 레포트로 발행**합니다. 모리가 얼마나 자랐는지, 사진·그림·텍스트 중 어떤 낙서가 많았는지, 이번 달 최고의 낙서까지 한눈에 보여줍니다.

```
📷 docs/screens/03_report.png
```

### 4. 환경설정

이름·상대방 별명, 그룹 배경 색상, 초대 코드, 알림 등 **간단한 설정**을 할 수 있습니다.

```
📷 docs/screens/04_settings.png
```

### 5. 낙서 보내기 — 펜

핵심 기능입니다. **펜의 종류(펜/지우개/형광펜), 색상, 굵기**를 조절할 수 있어요. 그 아래에는 **사라지는 낙서**를 보내는 버튼이 있습니다. 사라지는 낙서는 기록에 남지 않아 가벼운 장난을 칠 수 있죠.

```
📷 docs/screens/05_draw_pen.png
```

### 6. 사진 낙서 — 카메라 / 갤러리

하단 **갤러리 버튼**을 누르면 **카메라로 바로 촬영**하거나 **갤러리에서 사진을 선택**해, 사진 위에 낙서를 얹어 보낼 수 있습니다.

```
📷 docs/screens/06_gallery.png
```

### 7. 낙서 사진첩 — 목록 / 달력

세 번째 탭에서는 그동안 주고받은 낙서가 모입니다. **목록**으로 한눈에 볼 수도 있고, **달력**에서 날짜마다 어떤 낙서를 주고받았는지 확인할 수도 있어요. 낙서가 쌓이면 모리가 **AI 큐레이션 앨범**(‘귀여운 캐릭터와 낙서’, ‘밤과 빨간색의 조화’ 등)으로 자동으로 묶어줍니다.

```
📷 docs/screens/07a_album_list.png
```
```
📷 docs/screens/07b_album_calendar.png
```

### 8. 모리네 집 — 성장 & 그림체 학습

마지막 탭은 **모리네 집**입니다. 모리는 커플이 함께 키우는 캐릭터로, 낙서를 주고받거나 찌르기를 하면 조금씩 성장합니다. 또 하나의 핵심은 **그림체 학습** — 둘이 낙서를 주고받다 보면 모리가 **우리의 그림체를 학습**해 직접 그림을 그려 선물합니다. 벌어들인 코인으로 **모리를 꾸밀** 수도 있어요.

```
📷 docs/screens/08_pethouse.png
```

### 9. 모리의 그림 일기장

학습한 그림체로 모리가 그린 그림들을 **일기장**에서 날짜별로 모아 볼 수 있습니다.

```
📷 docs/screens/09_diary.png
```

### 10. 이웃집 놀러가기

마지막은 **이웃집 놀러가기**입니다. 랜덤으로 다른 커플이 모리를 어떻게 꾸몄는지 구경하고, **검색**으로 친구 커플의 집을 정확히 찾아갈 수도 있어요.

```
📷 docs/screens/10_neighbor.png
```

> 노션에서 준비한 바는 여기까지입니다. 동적인 기능이 많다 보니 **시연에서 더 자세히** 보여드리겠습니다.

## Phase 2. 라이브 시연 (휴대폰 2대)

정적인 화면으로 다 담기 어려운 **실시간성**을 직접 보여드립니다.

- **커플 1 (라이브, 빈 상태에서 시작)** — 그룹 생성 → 초대 코드로 가입 → 첫 낙서 왕복 → 사라지는 낙서 → 찌르기 → 모리 쓰다듬기까지 **처음부터 끝까지** 실시간으로.
- **커플 2 (성숙 커플, 경수 ♥ 수민)** — 그림체 학습이 끝난 상태에서 그림 일기·월간 레포트·꾸며진 모리를 보여줌.
- **이웃집** — 다른 커플들의 꾸며진 집을 방문.

> 두 커플은 앱 **환경설정의 ‘데모 계정 전환’**으로 한 대에서 오갈 수 있게 준비합니다.

## 발표 대본 (전문 · 이어읽기용)

> 안녕하세요, 커플 앱 ‘두드림’으로 돌아온 안종화, 이종혁입니다. 혹시 여기서 커플 앱을 사용해보신 분들 손 한번 들어주시겠어요? 네, 혹시 여러분 커플 앱을 왜 사용하는지 고민해보신 적이 있나요? 저도 이 프로젝트를 하면서 많이 고민해봤습니다. 다른 친구들에게 의견을 물어보기도 했는데요, 보통 답변이 두 가지 정도로 좁혀지더라고요. 첫번째, 둘만의 공간을 얻은 것 같은 기분을 느낍니다. 둘만의 일정이나 답변, 사진 등을 기록해두고 종종 들여다보면서 만족감을 느낀다는거죠. 두번째, 지속적인 소통의 기회가 됩니다. 썸원이 대표적으로 그렇죠. 매일 질문을 하니까 자꾸 대화를 하게 되고, 또 막 궁금하진 않았어도 알고 싶은 것을 알게 되잖아요. 그래서 저희는 이 점을 정확히 짚는 앱을 개발해보고 싶었습니다.
>
> 앱 내부 화면을 보여드리면서 설명드릴게요. 우선 처음 앱을 실행하시면 그룹을 생성하거나 가입할 수 있습니다. 상대방이 그룹에 참가하면 앱에 진입하실 수 있어요. 앱에 들어가면 처음 보실 화면은 이 화면입니다. 우선 눈에 들어오는건 저희 마스코트인 모리, 그리고 저희 앱 핵심 기능인 낙서 보내기 버튼입니다. 자세한 기능은 추후에, 그리고 시연 때 또 설명드리겠습니다. 이 화면은 월간 레포트입니다. 서로 주고받은 낙서들과 활동을 모아 레포트로 발행합니다. 환경설정에서는 간단한 설정들을 하실 수 있어요. 이제 핵심 기능 낙서 보내기를 살펴보겠습니다. 우선 간단하게 펜의 종류와 색상, 굵기를 조절하실 수 있어요. 그 아래에는 사라지는 낙서를 보낼 수 있는 버튼이 있습니다. 사라지는 낙서는 기록에 남지 않아서 간단한 장난을 칠 수 있겠죠? 하단 갤러리 버튼을 누르면 이렇게 사진을 찍거나 선택할 수 있는 버튼이 있어요. 세 번째 탭 낙서 사진첩에서는 그동안 주고받은 사진들이 모여있습니다. 이렇게 한눈에 보실 수도 있어요. 또, 달력에서 날짜마다 어떤 낙서를 주고받았는지도 확인할 수 있습니다. 마지막 탭은 모리네 집입니다. 모리는 커플이 함께 키우게 되는 캐릭터예요. 낙서를 주고받거나, 찌르기를 하면 캐릭터가 조금씩 성장합니다. 또 하나 핵심 기능은 그림체 학습이에요. 커플끼리 낙서를 주고받다 보면 모리가 그림체를 학습하고, 우리의 그림체로 그림을 그려 보여줍니다. 일기장에서는 그 그림들을 모아서 보실 수 있어요. 마지막은 이웃집 놀러가기 기능입니다. 랜덤으로 다른 커플들이 모리를 어떻게 꾸몄는지 보실 수 있고, 검색 기능을 이용해서 친구 커플의 집을 정확히 찾아갈 수도 있어요.
>
> 노션에서 준비한 바는 여기까지입니다. 동적인 기능들이 많다 보니 시연에서 더 자세히 설명드리겠습니다.
