# Memory Pager — 구현 상태 (정직판)

> 커플(2인) 전용 폐쇄형 소셜 앱. 사진·손낙서를 삐삐 감성으로 주고받고,
> 함께 키우는 펫이 낙서를 보고 그림 일기를 그려준다.
> Flutter(Android) + FastAPI + Socket.IO + MySQL + GPU(EXAONE LLM / Stable Diffusion).
>
> **이 문서는 2026-07-13 외부 감사 후 전면 교정한 것이다. 이전 판은 "엔드포인트 배선 + happy-path
> E2E"만 보고 ✅를 과장했다.** 상태는 실제 동작 기준으로 다시 매겼다.
>
> 상태: ✅ 실서버 동작 검증 · 🟡 부분/제약 있음 · 🔴 미동작(정적 UI/미구현).

---

## ⚠️ 먼저 — 반드시 알아야 할 것
1. **release APK는 방금 전까지 서버 통신 불가였다.** `main/AndroidManifest.xml`에 INTERNET 권한이
   없어 debug/profile에만 있었다. → **수정 완료.** 단 **기존 CI 아티팩트(그 이전 커밋)는 폐기**하고,
   INTERNET 권한이 포함된 **새 APK를 다시 빌드**해야 한다.
2. **백엔드와 프론트가 다른 브랜치였다.** 질문 API·join 응답 래핑은 `anjonghwa` 브랜치에만 있었다
   (배포 서버는 anjonghwa라 라이브는 됐음). → **anjonghwa를 front-remake로 머지 완료.**
   이제 front-remake 하나로 전체 시스템 재현 가능. (최종적으로 main 통합은 별도 결정 필요)
3. **실기기 검증은 아직 안 됐다.** 헤드리스 구동이 막혀 웹 미리보기 + API E2E로만 검증했다.
   갤럭시에 새 APK를 깔고 2인 실사용 확인이 남았다.

## 백엔드 (배포 서버, anjonghwa=front-remake) ✅ — 거의 완성
- 인증·기기 등록, 그룹 생성/참여/상세/수정, 멤버 별명.
- 낙서(그림/텍스트/사진) 업로드·조회·확인(view)·사라지기(ephemeral)·답장(parent_id).
- 펫 조회/쓰다듬기, 찌르기, 오늘의 질문(E-1), 그림일기 목록, 월간 레포트(+생성), 홈 위젯.
- Socket.IO `/rt` 실시간 이벤트.
- GPU 연동: 낙서→일기 이미지(SD), 펫 활동/대사(LLM), 레포트.
- 통합 테스트 다수 그린.

## 프론트 화면별 상태
| 화면 | 상태 | 메모 |
|---|---|---|
| 온보딩(이름/그룹/별명) | 🟡 | 실서버 등록·생성·참여 동작. **실서버 흐름에서 별명 단계 스킵**(생성자는 상대가 없어 애매). 그룹/펫 이름 입력 UI 자체가 디자인에 없음(기본값 사용) |
| 홈 | ✅ | 펫 상태·활동표정·모자, 찌르기(상대 있을 때 소켓 전송), 쓰다듬기, 오늘의 질문(엔터 제출) |
| 캔버스(그리기) | 🟡 | 그림 전송·사라지기·답장 동작. **사진 버튼은 카메라/갤러리 아님**(고정 샘플 토글, 전송 PNG에 배경 미포함) 🔴. **텍스트 낙서 전송 UI 없음**(API만 존재) 🔴 |
| 앨범 | 🟡 | 타임라인·인물필터·격자갤러리·달력진입 동작. **주간 스트립/배너는 정적** 🔴 |
| 뷰어 | 🟡 | 이미지·찌르기·답장 동작. **하트 버튼 무동작** 🔴. 낙서 AI 캡션은 **서버 DoodleOut이 노출 안 함**(백엔드 갭) 🔴 |
| 그림일기 | ✅ | 실 일기(서버 이미지) 로드·빈 상태·diary:new 갱신 |

| 펫하우스/스토어 | 🟡 | 펫·배경색·일기/이웃 진입. **스토어 코인 게이팅 미구현**(U-1), **카테고리/저장 버튼 무동작** 🔴 |
| 이웃집 | 🔴 | 목데이터. **검색 무동작** |
| 설정 | 🟡 | 별명/이름 편집(엔터 저장), 배경색, **초대코드 복사(수정됨)**. **로그아웃은 토큰/그룹 탈퇴 없이 화면만 전환** 🔴 |
| 달력 | 🔴 | 월 이동·날짜 선택 정적 UI |
